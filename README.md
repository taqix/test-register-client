# aplikacja kliencka łącząca się z [serwerem](https://github.com/taqix/test-register-server)
aplikacja napisana w frameworku react native \
służąca do prostego logowania się na serwerze lokalnym \
żeby włączyć aplikację musimy być połączeni do jednej sieci, serwer i klient \
w pliku Settings.json trzeba ustawić swoje lokalne ip \
żeby odpalić klienta musimy użyć [node.js](https://nodejs.org/en) oraz komend w terminalu \
```
npm i
npm start
```
w terminalu wyświetli nam się kod qr który musimy zeskanować w mobilnej aplikacji [expo](https://expo.dev/)
